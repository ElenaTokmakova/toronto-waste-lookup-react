
## Features

* React 16
* ReactStrap
* React Font Awesome
* SASS
* Autoprefixer
* Webpack 4
* Babel
* Hot Module Replacement

## Installation

* clone the repository
* npm install
* npm dev
* visit `http://localhost:8080/`

## View Production Build

* clone the repository
* npm install
* npm start
* visit `http://localhost:3000/`
